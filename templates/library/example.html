Exemple d'introduction à l'utilisation de l'assistant de preuve

Pour comprendre comment fonctionne l'assistant de preuve, allez à la section Demonstrate, tapez new(); dans le formulaire puis cliquez sur le boutton 'compiler'. Normalement, vous venez de voir apparaitre le séquent 1 : gamma; |- True. Vous venez de créer le séquent avec aucune hypothèse et qui a pour conclusion la proposition 'True'. Vous pouvez maintenant, par exemple, vouloir ajouter l'hypothèse 'A implies B' à votre séquent : tapez addhyp(1, A implies B); . Enfin, tapez axiomintro(2, 0); pour passer cette hypothèses en conclusion de votre séquent. Pour aller plus loin, allez voir dans la section documentation pour connaître toutes les instructions disponibles.
